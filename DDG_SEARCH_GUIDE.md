## 🚀 快速开始

### **基础用法**

```bash
# 进入项目目录
cd ~/Desktop/VPT-初诊数据

# 普通搜索（返回5条结果）
python3 ddg_search.py "搜索问题"

# 指定结果数量
python3 ddg_search.py "搜索问题" 3

# 新闻搜索
python3 ddg_search.py "新闻关键词" 5 news
```

### **使用示例**

```bash
# 搜索产品信息
python3 ddg_search.py "iPhone 16 Pro 价格"

python3 ddg_search.py "2026年科技新闻" 5 news

# 英文搜索
python3 ddg_search.py "Python 3.14 release notes"
```

## 🎨 使用场景

### **最适合 DuckDuckGo 的场景：**

✅ **日常信息查询**
- 产品价格、参数
- 技术文档、教程
- 新闻资讯
- 百科知识

✅ **Claude WebSearch 达到上限时**
- 作为补充方案
- 无限次免费使用

✅ **注重隐私时**
- 不记录搜索历史
- 不创建用户画像

✅ **需要验证信息时**
- 多源对比
- 交叉验证

### **不适合的场景：**

❌ **英文学术文献**
- 建议用 Claude WebSearch + PubMed

❌ **需要高质量摘要时**
- DuckDuckGo 返回原始结果，无AI总结
