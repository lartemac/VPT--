## 💡 实用技巧

### **1. 提高搜索准确度**
```bash
# 具体化问题
python3 glm_search.py "中信银行2026年留学生白金信用卡年费及减免政策"

# 而不是
python3 glm_search.py "信用卡年费"
```

### **2. 使用详细程度控制输出长度**
```bash
# 简洁回答（一句话）
python3 glm_search.py "中信银行留学卡年费" 简洁

# 详细回答（包含背景和细节）
python3 glm_search.py "中信银行留学卡年费" 详细
```

### **3. 批量搜索（Shell 循环）**
```bash
#!/bin/bash
questions=(
  "中信银行留学卡年费"
  "招商银行信用卡年费"
  "广发银行全币种卡年费"
)

for q in "${questions[@]}"; do
  echo "搜索：$q"
  python3 glm_search.py "$q"
  echo "---"
done
```

## ⚠️ 注意事项

1. **API Key 安全**
   - API Key 已内置在脚本中
   - 不要将脚本上传到公开仓库

2. **使用成本**
   - 你已订阅 GLM-4，无额外费用
   - 但仍建议合理使用，避免浪费

3. **搜索质量**
   - 中文搜索优秀
   - 英文搜索可能不如 Claude
   - 学术文献建议用 Claude + PubMed

4. **月度重置**
   - Claude WebSearch 每月1日重置（2026-02-01 下次重置）
   - 重置后可以继续使用 Claude 搜索
